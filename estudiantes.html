Promedio de Notas para N Estudiantes

Calcula el promedio de dos notas para una cantidad N de estudiantes. Ingresa la cantidad de estudiantes y luego las notas de cada uno.

Comenzar Volver al Inicio

Configuración Inicial

Ingresa la cantidad de estudiantes para generar el formulario

¿Cómo funciona?

Paso 1

Ingresa la cantidad de estudiantes para los que deseas calcular promedios.

Paso 2

Completa las dos notas para cada estudiante en los campos generados automáticamente.

Paso 3

Haz clic en "Calcular Promedio" para cada estudiante y ver los resultados.
